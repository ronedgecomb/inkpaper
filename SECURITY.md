# Security policy

## Supported versions

The 0.1.x line will receive security fixes after the initial release. When a newer line replaces it, this table will be updated explicitly.

| Version | Supported |
| --- | --- |
| 0.1.x | Yes, after 0.1.0 is published |
| Earlier | No |

## Reporting a vulnerability

Do not open a public issue for a suspected vulnerability. Use [GitHub private vulnerability reporting](https://github.com/ronedgecomb/inkpaper/security/advisories/new).

This is a single-maintainer open-source project. Reports are reviewed and handled on a best-effort basis without a guaranteed response deadline. Include affected versions, reproduction steps, impact, and any suggested mitigation. Coordinated disclosure is appreciated.
